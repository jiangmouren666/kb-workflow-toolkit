# Inbox

把未整理资料先放到这里，包括论文 PDF/链接、网页摘录、临时想法、回测结果和外部资料。

## 入库规则

1. 不在 inbox 里长期维护正式知识。
2. 每条资料先判断类型：`paper`、`method`、`standard`、`case`、`backtest`。
3. 使用 `knowledge-ingest` 整理后，再移动或改写到正式目录。
4. 对量化资料，必须标记未来函数、数据口径、样本区间、交易成本等风险。
